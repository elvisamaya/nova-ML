# 🚀 AstroML

**AstroML Journey** is an interactive, story-driven web app that teaches the fundamentals of machine learning through a space-themed adventure.

Instead of traditional lessons, users travel across different “worlds,” each representing a key concept in machine learning — from data to training to evaluation — guided by Captain Nova.

---

## 🌌 Overview

In this experience, users:

- Explore **6 unique worlds**, each representing a step in the ML pipeline
- Learn concepts through **narrative + interactive checkpoints**
- Complete **missions (quizzes)** to unlock the next stage
- Progress through a **game-like journey** rather than static content

---

## 🧠 Concepts Covered

Each world corresponds to a core ML idea:

1. **Data World** – What training data is  
2. **Augmentia** – Data augmentation  
3. **Brain Forge** – Neural networks & layers  
4. **Training Arena** – Loss & learning from mistakes  
5. **Test Zone** – Generalization & evaluation  
6. **Insight Galaxy** – Tracking performance (loss/accuracy)

---

## 🎮 Features

- 🎨 **8-bit retro UI** with pixel art styling  
- 🌍 **Dynamic world progression system** (locked/unlocked stages)  
- 🧩 **Interactive quizzes** (must answer correctly to proceed)  
- 📈 **XP + progress tracking system**  
- 🪐 **Dynamic planet visuals** that change per world  
- 🧑‍🚀 Custom **pixel Captain Nova character**  
- 🏁 **Final completion screen** after finishing all stages  

---

## 🛠️ Tech Stack

- **React** (frontend framework)
- **Vite** (fast development server + build tool)
- **JavaScript (ES6+)**
- **Custom CSS (no UI frameworks)**

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/astro-ml-journey.git
cd astro-ml-journey
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```

Then open:
```
http://localhost:5173
```

## 🎯 Purpose

This project explores how **education + storytelling + interactivity** can improve engagement in technical topics like machine learning.

It was designed to:
- Make ML concepts accessible to beginners
- Replace passive learning with active exploration
- Combine **game design + education + UI engineering**
